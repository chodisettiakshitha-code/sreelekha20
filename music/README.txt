╠══════════════════════════════════════════════════════════════╣
║                                                              ║
║  BACKGROUND MUSIC:                                           ║
║  1. Place your audio file here (e.g., birthday-song.mp3)     ║
║  2. Open js/main.js                                          ║
║  3. Set BACKGROUND_MUSIC_FILE = 'music/birthday-song.mp3'   ║
║  4. The floating play/pause button will now work!            ║
║                                                              ║
║  PLAYLIST SONGS:                                             ║
║  1. Place song files here                                    ║
║  2. Add playlist tracks in index.html (see comments there)   ║
║     using this template:                                     ║
║                                                              ║
║  <div class="playlist-track" data-src="music/song.mp3">     ║
║      <div class="track-number">1</div>                      ║
║      <div class="track-info">                                ║
║          <span class="track-name">Song Title</span>         ║
║          <span class="track-artist">Artist Name</span>      ║
║      </div>                                                  ║
║      <div class="track-duration">3:45</div>                 ║
║  </div>                                                      ║
║                                                              ║
║  Supported formats: .mp3, .wav, .ogg, .m4a                  ║
